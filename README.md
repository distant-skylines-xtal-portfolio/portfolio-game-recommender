# portfolio-game-recommender
Webapp game recommender with parameters. Type in various tags (pulled from IGDB.com) and find games
matching those tags. Backend is built into the portfolio-backend project, using axios to create
a custom API which feeds the data to this project which acts a front-end.

Tech used: 
- React
- React Router
- TypeScript 
- Axios
- Framer motion
- Vite

Currently hosted at: [evanmclay.dev/apps/game-recommender/](https://evanmclay.dev/apps/game-recommender/)

To run a local dev server, you will need to clone both this project and the portfolio-backend project.
Then with both projects run <br />
npm run dev
